TeX は、コンピュータ上で組版処理を行うシステムです。この記事では、TeX を自分の PC にインストールした際の、実際に行ったインストールの手順やインストールしてみた感想について述べます。TeX のインストール方法の詳細についての解説記事ではありません。

## TeX とは
TeX は「組版処理システム」と呼ばれます。簡単に言えば (正確ではありませんが)、文章を記述したテキストファイルを、TeX に含まれるプログラムに渡すと、文字やレイアウトがきれいに整形された PDF ファイルが出力されるというものです。

Web サイト「ウィキペディア」の記事によると、TeX の開発は1978年から始まり、現在も継続しています。主な特徴として、無料で配布されていること、数式の組版の美しさなどがあります。このような特徴から、数式を用いる論文、レポートの作成、受け渡しの場面において、世界的に利用されています。

知識の表現、伝達、保存の手段として今後、この地位が維持されるのかどうかは気になるところです。


## インストール環境
この記事では、OS として Windows 11 Home を搭載した PC へのインストールについて述べます。

Windows PC の場合、WSL (Windows Subsystem for Linux) 上の Linux へインストールするという選択肢もありますが、こちらについては述べません。


## 情報収集
インストールに際して、まず Google 検索で「tex」、「インストール」などのキーワードで検索し、検索結果一覧の上位に表示された Web サイト「[TeX Wiki](https://texwiki.texjp.org/)」で、インストールに関する情報を得ることができました。

この Web サイト上の TeX のインストールに関する記事の中で、以下の2つの Web サイトが紹介されています。

- [TeX Users Group (TUG)](https://www.tug.org/)
- [Comprehensive TeX Archive Network (CTAN)](https://www.ctan.org/)

これらの Web サイトでは、TeX のインストールに必要なプログラムをダウンロードすることができます。


## インストール
前節で紹介した Web サイトで得られた情報にしたがって、インストールを行っていきます。

TeX は、「ディストリビューション」という形態で配布されていて、また、TeX のディストリビューションは複数存在しています。インストールに際しては、まず、それらの中から1つを選ぶことから始まり、その後は、そのディストリビューション固有のインストール方法にしたがってインストールする流れになります。

1. ディストリビューションの選定
1. インストール方法の選定
1. インストールオプションの選定


### ディストリビューションの選定
ディストリビューションとは、TeX を利用するために必要な様々なプログラムを組み合わせたものです。組み合わせ方の違いによって複数のディストリビューションが存在します。私は、「TeX Live」というディストリビューションを選びました。以下では、TeX Live のインストールについてのみ述べます。


### インストール方法の選定
TeX Live のインストール方法は、以下の2つがあります。

- ネットワークインストーラ (install-tl-windows.exe) を用いる方法
- ISO イメージ (texlive2024.iso 等) を用いる方法

ネットワークインストーラを用いる方法は、TeX Wiki でも TeX Users Group でも推奨されている方法です。私は最初、こちらの方法でインストールを試みました。ところが、私のネット環境では、インストールオプションを変更して構成をいくらか軽量にした程度では、インストールにかかる時間が相当長くなりそうでした。

そこで、ISO イメージを用いる方法に切り替えることにしました。こちらの方法でも、ISO イメージのサイズは数Gバイトあるので、ダウンロードに時間がかかるという点では同じです。しかし、ISO イメージのダウンロード作業とファイルの展開作業とが分離されるので、一度ダウンロード作業が完了してしまえば、ファイルの展開作業を何らかの理由でやり直す場合に、ダウンロード作業からやり直す必要はなく、ファイルの展開作業から再開すれば良いという点は利点だと思います。

ISO イメージのダウンロードは、TeX Users Group や CTAN のトップページから「Download」や「TeX Live」の語を頼りに探すと見つかるリンクの先のページでできます。ダウンロードした ISO イメージに含まれているファイル install-tl-windows.bat を実行すると TeX Live のインストーラが起動します。


### インストールオプションの選定
TeX Live のインストーラでは、様々なオプションを指定してインストールの構成をカスタマイズすることができます。

私は最初、不要と思われるオプションを外し、インストールの構成を軽量にしてインストールを行いました。ところが、過去に別の PC で作成した TeX ファイルを、今回新たにインストールした TeX で再度処理しようとすると、エラーが発生してしまいました。理由は、処理に必要な機能に対応するインストールオプションを、インストール時に指定していなかったためです。このような場合、TeX にはインストール後に機能を追加する仕組みがあるので、エラーの原因となった機能を追加することで対処できます。しかし、私のような TeX 初心者にとっては、どの機能がどのオプションに対応しているのかを判断することが難しいのです。

結局 TeX を、全てのインストールオプションを指定して再インストールするという力技で対処することにしました。こうすることで、将来遭遇するかも知れない、上で述べたようなエラーを未然に予防することができます。その代償として、TeX に関する知識を深める機会が減ってしまうことは欠点だと思います。

以上が、私が TeX のインストールに際して、実際に行った手順とその感想です。


## まとめ
- インストール方法に関する情報は TeX Wiki で取得した。

- インストール方法は ISO イメージを用いる方法を採用した。
- インストールオプションは全て指定した。
