Homebrew は macOS 用のパッケージマネージャーとして有名ですが、Linux 用も配布されています。
Ubuntu は Linux ディストリビューションの1つです。
この記事は、Linux 用 Homebrew を Ubuntu へインストールした際の個人的なメモです。


## Homebrew をインストールする動機
Homebrew はパッケージマネージャーの1つです。

パッケージマネージャー、またはパッケージ管理システムとは、様々なソフトウェアのインストール/アンインストール、及びアップデートを、個々のソフトウェアに依らない統一的な方法で、また自動的に行うための仕組みであり、また、その仕組みを実現するソフトウェアのことです。
パッケージマネージャーを使用しないでソフトウェアをインストールすることは可能ですが、ソフトウェアによってインストール方法がまちまちで、管理が煩雑になります。

Ubuntu においては、パッケージ管理システムとして APT が標準的に用いられています。
私は最初、この APT を用いてソフトウェア (Node.js や Ruby など) をインストールしようと考えていました。しかし、APT でインストールすると最新のバージョンではなく、やや古いバージョンがインストールされてしまうことに問題を感じていました。
バージョンが古いと、それに依存する別のソフトウェア (ライブラリやフレームワークなど) が動作しないことがあるからです。

Homebrew であれば、バージョンを指定してソフトウェアをインストールできるようでしたので、試してみることにしました。

尚、上で挙げた Node.js、Ruby に関しては、Node.js に対して nvm、Ruby に対して rbenv などのバージョン管理ソフトウェアが存在します。これらを使用して、最新のバージョンの Node.js、Ruby をインストールすることが可能です。
しかし、これらのバージョン管理ソフトウェア自体のバージョンを管理する必要が生じ、結局、堂々巡りになってしまいます。


## 不安な点
Homebrew は、ソフトウェアをインストールする際、formula と呼ばれるインストール方法が記述されたスクリプトに従ってインストールを行います。この formula は誰でも作成、公開することができます。ソフトウェアをインストールする前に、formula の作成者が信頼できるかどうか確認する必要があります。

また、formula の作成、公開は、有志の手に委ねられているため、ユーザーの少ないソフトウェアの場合、formula が公開されておらず、Homebrew でインストールできないことがあります。

## インストール
インストールの流れは以下の通りです。

1. インストーラーのダウンロードと実行
1. シェルの設定変更
1. 設定の反映


### インストーラーのダウンロードと実行
公式サイトに従って、以下のコマンドを実行します。
```bash
$ /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```
`curl`コマンドで Homebrew をインストールするためのシェルスクリプト`install.sh`をダウンロードし、それを実行しています。

このシェルスクリプトの実行には、いくつかの開発用ソフトウェアが必要です。私の使用している Ubuntu のバージョンでは、既にインストールされていました。インストールされていない場合は、以下のコマンドでインストールできます。
```bash
$ sudo apt-get install build-essential procps curl file git
```


### シェルの設定変更
シェルスクリプト`install.sh`を実行すると表示されるメッセージに従って、以下のコマンドを実行します。
```bash
$ echo >> /home/ユーザー名/.bashrc
$ echo 'eval "$(/home/linuxbrew/.linuxbrew/bin/brew shellenv bash)"' >> /home/ユーザー名/.bashrc
```
1行目のコマンドでは、シェルの設定ファイル`.bashrc`の末尾に空行を1行追記しています。
2行目のコマンドでも同様に、`.bashrc`の末尾に`eval ...`を追記しています。`brew shellenv bash`で環境設定用のコマンドを出力し、`$(...)`で囲むことにより、それを文字列化します。そして、この文字列を`eval`に渡して実行させています。


### 設定の反映
設定を反映させるために、以下を実行するか、ターミナルを再起動します。
```bash
$ source ~/.bashrc
```
以上で、インストールは完了です。
